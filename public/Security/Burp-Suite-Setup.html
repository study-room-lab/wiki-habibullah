The "Man-in-the-Middle"

Burp Suite operates on a simple premise: Trust Logic. By telling your browser to trust Burp as a Certificate Authority (CA), you can break SSL/TLS encryption. You see the raw HTTP requests before they leave your machine, and the raw responses before the browser renders them.

2. Browser Integration (The Critical Step) üîå

Do not configure your system-wide proxy. That breaks Spotify and Discord. Use Firefox with FoxyProxy.

Step A: FoxyProxy Setup

  1. Install the FoxyProxy Standard extension in Firefox.

  2. Add a new proxy:

    • Type: HTTP

    • IP: 127.0.0.1

    • Port: 8080

    • Color: Green (or whatever signals ‚ÄúHacking Mode‚Äù)

  3. Click the FoxyProxy icon and select your new profile.

Step B: Installing the CA Certificate

If you skip this, HTTPS sites will throw ‚ÄúSecure Connection Failed‚Äù errors.

  1. Ensure Burp is running and FoxyProxy is set to Burp.

  2. Visit http://burp in Firefox.

  3. Click ‚ÄúCA Certificate‚Äù in the top right to download cacert.der.

  4. In Firefox: Settings ‚Üí Search ‚ÄúCertificates‚Äù ‚Üí View Certificates.

  5. Click Import ‚Üí Select cacert.der.

  6. Check both boxes:

    • Trust this CA to identify websites.

    • Trust this CA to identify email users.

3. The Arsenal: Core Tools üõ†Ô∏è

A. Proxy (The Gatekeeper)

  • Intercept: Toggles traffic flow.

    • On: Requests hang until you click ‚ÄúForward‚Äù. Great for modifying a request on the fly.

    • Off: Traffic flows passively. Use this 90% of the time and check HTTP History.

  • HTTP History: The log of everything. Filter by ‚ÄúScope‚Äù to hide Google Analytics noise.

B. Repeater (The Lab)

Shortcut: Ctrl + R (Send to Repeater)

This is where research happens. You take a request, tweak one parameter (e.g., change id=1 to id=1‚Äô), and hit Send to see the raw response.

  • Workflow: Send ‚Üí Modify ‚Üí Check Response Code/Length ‚Üí Repeat.

C. Intruder (The Machine Gun)

Shortcut: Ctrl + I

Used for brute-forcing and fuzzing.

  1. Payload Positions: Highlight the data you want to fuzz (e.g., a password field).

  2. Attack Type:

    • Sniper: Single payload set. Tries list A against slot 1, then slot 2.

    • Battering Ram: Same payload in all slots simultaneously.

    • Pitchfork: Different payloads for different slots (User/Pass lists).

D. Decoder

A built-in utility for Base64, URL, HTML, and Hex encoding.

  • Tip: Use the ‚ÄúSmart Decode‚Äù button if you see garbage text.

4. Methodology: The Search for Anomalies

When analyzing a target, I follow this loop:

  1. Map the App: Click every button. Fill every form. Populate the Site Map.

  2. Define Scope: Right-click the domain in Target tab ‚Üí ‚ÄúAdd to Scope‚Äù.

    • Proxy Filter: Click the filter bar ‚Üí ‚ÄúShow only in-scope items‚Äù.

  3. Hunt:

    • Look for ID parameters (user=101) ‚Üí Test IDOR.

    • Look for search bars ‚Üí Test XSS/SQLi.

    • Look for file uploads ‚Üí Test RCE.

5. Pro Tips for Fedora Users

  • Dark Mode: User Options ‚Üí Display ‚Üí Look and Feel ‚Üí Darcula.

  • Font: Change HTTP Message font to JetBrains Mono size 14 for readability.

  • Extensions (BApp Store):

    • Turbo Intruder: For high-speed race condition testing.

    • Logger++: Better logging than the default history.

    • JSON Web Tokens (JWT) Editor: Essential for modern API hacking.
